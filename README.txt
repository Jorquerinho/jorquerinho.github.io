# Juego de Ajedrez Online - Proyecto Semestral

Plataforma web para jugar ajedrez en línea. Incluye registro, login, partidas en curso, historial y estadísticas del jugador.

## Características
- Vista del Registro y autenticación de usuarios
- Vista Interfaz
- VIsta tablero de ajedrez
- Vista Estadísticas de partidas y perfil de usuario


## Tecnologías utilizadas
- HTML
- CSS

# Validaciones
- HTML validado por la extension HTMLHint
-CSS validado por la extension Stylelint
